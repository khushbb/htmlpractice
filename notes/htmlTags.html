HTML Tags

  • <html>

    • This tag tells the browser the start of the page

  • <head>

    • This tag designates the main header

  • <title>

    • This tag displays a title in the main header

  • <style type=" ">

    • This tag sets the style of the whole page

  • <body>

    • This tag designates the substance of the page

  • <h#>

    • This tag denotes a headline, the lower the number the bigger the word size

  • <p id=" ">

    • This tag denotes a paragraph, also can attach an id to it

  • <q>

    • This tag denotes a quote

  • <blockquote>

    • This tag denotes a blockquote

  • <ul>

    • This tag designates the start of an unordered list

  • <ol>

    • This tag designates the start of an ordered list

  • <li>

    • This tag denotes an item in the list

  • <img src=" ">

    • Diplays image from a source file

  • <br>

    • Denotes a line break

  • <em>

    • Denotes the following text to be emphasised

  • <a href=" " title=" " target=" ">

    • Designates the following text as a link, title will display when the mouse hovers over the link, and target="_blank" will always open the link in a new window

  • <time>

    • Designates a date and/or time

  • <pre>

    • Displays following text exactly as typed/formatted

  • <code>

    • Displays code from computer program

  • <strong>

    • Displays following test strongly emphasised

  • Special Characters

    • Look up abbreviations such as &
